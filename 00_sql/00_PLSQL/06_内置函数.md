## 转换函数
- TO_DATE（字符串，格式）:
  + TO_DATE('2015-JAN-15', 'YYYY-MON-DD');
  + 输出：1 / 15 / 2015
- TO_NUMBER（文本，格式）:
  + TO_NUMBER('1234′,'9999')；
  + 输出：1234
 
## 字符串函数
- INSTR(文本，字符串，开始，发生):
  + INSTR(‘AEROPLANE’,‘E’,2,1)
  + 输出：2
- SUBSTR（文本，开始，长度）:
  +  substr('飞机',1,7)
  + 输出: 航空飞机
- LPAD（文本，长度，pad_char）:
  +  LPAD('guru99', 10, '$')；
  + 输出：$$$$guru99
- RPAD（文本，长度，pad_char）:
  +  RPAD('guru99′,10,'-')
  + 输出：guru99----
- LTRIM（文本）,RTRIM（文本）

## 日期函数
- TRUNC : 将日期变量四舍五入为较低的可能值
  + TRUNC(sysdate)；
  + 输出：10 年 4 月 2015 日下午 2:12:39
