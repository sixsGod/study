## 变量
- 声明变量的语法
```sql
-- 可以在声明变量的同时给变量强制性的加上 NOT NULL 约束条件，此时变量在初始化时必须赋值。
Variable_name [CONSTANT] databyte [NOT NULL][:=|DEFAULT expression]
```

- 变量赋值
```sql
-- 1.直接给变量赋值
X:=200;
Y=Y+(X*20);

-- 2.通过 SQL SELECT INTO 或 FETCH INTO 给变量赋值
SELECT SUM(SALARY),SUM(SALARY*0.1)
INTO TOTAL_SALARY,TATAL_COMMISSION
FROM EMPLOYEE
WHERE DEPT=10; 
```

## 常量
- 声明常量的语法
```sql
-- 必须包括关键字 CONSTANT
ZERO_VALUE CONSTANT NUMBER:=0; 
```

## 数据类型
- 标量(scalar)数据类型
  + Numeric:数字数据类型
    * BINARY_INTEGER(用于存储单字节整数。要求存储长度低于 NUMBER 值。)
    * PLS_INTEGER(与 BINARY_INTEGER 基本相同，但采用机器运算时，PLS_INTEGER 提供更好的性能 )
    * NUMBER(存储数字值，包括整数和浮点数。可以选择精度和刻度方式，语法：number[（[,]）]。)
  + character:字符数据类型
    * CHAR:如果长度没有确定，缺省是 1 
    * LONG:存储可变长度字符串(只支持顺序访问方式)
    * RAW:用于存储二进制数据和字节字符串
    * LONGRAW:与 LONG 数据类型相似，同样他也不能在字符集之间进行转换
    * ROWID:与数据库 ROWID 伪列类型相同，能够存储一个行标
    * VARCHAR2:存储可变长度的字符串
  + date/time:01/01/4712 BC
  + boolean:TRUE/FALSE

- LOB 数据类型
LOB(大对象,Large object) 数据类型用于存储类似图像，声音这样的大型数据对象.最大长度不超过 4G,
在 PL/SQL 中操作 LOB 数据对象使用 ORACLE 提供的包 DBMS_LOB.LOB 数据类型可分为以下四类
  + BFILE
  + BLOB
  + CLOB
  + NCLOB
