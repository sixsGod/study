## 控制结构

1. IF语句
```sql
IF condition1 THEN
    statement1;
ELSIF condition2 THEN
    statement2;
ELSIF condition3 THEN
    statement3;
ELSE
    statement4;
END IF;
statement5; 
```


## 循环结构
1. LOOP 语句
```sql
-- LOOP 和 END LOOP 之间的语句无限次的执行,必须使用 EXIT 语句，强制循环结束
X:=100;
LOOP
X:=X+10;
IF X>1000 THEN
 EXIT;
END IF
END LOOP;
Y:=X;

X:=100;
LOOP
X:=X+10;
EXIT WHEN X>1000;
X:=X+10;
END LOOP;
Y:=X
```

2. WHILE..LOOP语句
```sql
-- 如果条件为 TRUE,则执行循环体内的语句，如果结果为 FALSE,则结束循环
X:=100;
WHILE X<=1000 LOOP
    X:=X+10;
END LOOP;
Y=X; 
```

3. FOR...LOOP语句(FOR 循环的循环次数是固定的)
```sql
-- FOR counter IN [REVERSE] start_range....end_range LOOP
--counter 是一个隐式声明的变量，他的初始值是 start_range
-- 如果使用了 REVERSE关键字，那么范围将是一个降序

X:=100;
FOR v_counter in 1..10 loop
    x:=x+10;
end loop
y:=x; 
```
