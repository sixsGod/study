## input输入

- ACCEPT
```cobol
ACCEPT USER-INPUT.
DISPLAY "入力された名前: ", USER-INPUT.

ACCEPT 変数名 FROM DATE.  *> YYYYMMDD
ACCEPT 変数名 FROM TIME.  *> HHMMSS
```

## output输出
1. DISPLAY文
```cobol
DISPLAY "Hello, World!".
DISPLAY "変数Aの値: " WS-A.
DISPLAY "変数Bの値: " WS-B, "変数Cの値: " WS-C.
```

2. WRITE文
```cobol
WRITE レコード名 [FROM データ項目].

FILE-CONTROL.
    SELECT OUTPUT-FILE ASSIGN TO "output.txt".
DATA DIVISION.
    FILE SECTION.
    FD OUTPUT-FILE.
    01 OUTPUT-RECORD PIC X(80).
    WORKING-STORAGE SECTION.
    01 WS-DATA PIC X(80) VALUE "This is a test record.".
PROCEDURE DIVISION.
    OPEN OUTPUT OUTPUT-FILE.
    MOVE WS-DATA TO OUTPUT-RECORD.
  * write record
    WRITE OUTPUT-RECORD.
    CLOSE OUTPUT-FILE.
    STOP RUN.
```
