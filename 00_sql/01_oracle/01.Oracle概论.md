## SQL*Plus工具
在输入Oracle数据库命令时，其关键字不区分大小写（比如，输入sqlplus或SQLPLUS都可以），但参数区分大小写。
```sql
-- @connect_identifier：表示连接的全局数据库名，若连接本机上的默认数据库，则可以省略。
SQLPLUS username[/password][@connect_identifier] [AS SYSOPER|SYSDBA]

connect system/1qaz2wsx as sysdba;
```

- 启动数据库实例
```sql
STARTUP [nomount | mount | open | force] [resetrict] [pfile=filename]
-- nomount：表示启动实例不加载数据库。
-- mount：表示启动实例、加载数据库并保持数据库的关闭状态。
-- open：表示启动实例、加载并打开数据库，这个是默认选项。
-- force：表示终止实例并重新启动数据库。
-- resetrict：用于指定以受限制的会话方式启动数据库
-- pfile：用于指定启动实例时所使用的文本参数文件，filename就是文件名。
若在startup语句中没有指定pfile参数，则Oracle首先读取默认位置的服务器初始化参数文件spfile，
若没有找到默认的服务器初始化参数文件，则将读取默认位置的文本初始化参数文件。
```

- 关闭数据库实例
```sql
SHUTDOWN [normal | transactional | immediate | abort]
-- transactional表示在当前所有的活动事务被提交完毕之后，关闭数据库。
-- abort表示以终止方式来关闭数据库。
```

## SQL语言的分类
1. 数据查询语言（DQL）
   - SELECT
2. 数据操纵语言（DML）
   - INSERT、UPDATE和DELETE
3. 事务控制语言（TCL）
   - COMMIT、ROLLBACK和SAVEPOINT 
4. 数据定义语言（DDL）
   - CREATE TABLE,ALTER TABLE,DROP TABLE
5. 数据控制语言（DCL）
   - GRANT和REVOKE
