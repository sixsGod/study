## EXPDP和IMPDP概述
数据泵导出使用服务器端的工具EXPDP将数据库对象的元数据（对象结构）或数据导出到转储文件中，而数据泵导入则是使用工具IMPDP将转储元件中的元数据及其数据导入到Oracle数据库中。

## EXPDP导出数据
```sql
-- 导出表(TABLE)
expdp scott/1qaz2wsx directory=dump_dir dumpfile=tab.dmp tables=emp,dept
-- 导出模式(SCHEMA)象
expdp system/1qaz2wsx directory = dump_dir dumpfile=schema.dmp schemas = scott,hr
```

## EXPDP命令参数
- CONTENT：content={all | data_only | metadata_only(对象定义)}
- QUERY：query=[schema.] [table_name:] query_clause
- DIRECTORY：directory=directory_object用于指定目录对象的名称
- DUMPFILE：dumpfile=[directory_object:]file_name[,...]
- FULL：full={y | n} 数据库导出  必须具有EXP_FULL_DATABASE角色或DBA角色
- LOGFILE：logfile=[directory_object:]file_name
- STATUS：status=integer 每隔特定时间间隔会显示作业完成的百分比
- TABLES：Tables=[schema_name.]table_name[:partition_name][,...]
- TABLESPACES：用于指定要导出表空间列表

## IMPDP导入数据
```sql
-- 导入表
impdp system/1qaz2wsx directory=dump_dir dumpfile=tab.dmp tables=scott.dept,scott.emp remap_schema=scott:system
-- 导入模式
impdp system/1qaz2wsx directory=dump_dir dumpfile=schema.dmp schemas=scott remap_schema=scott:system;
-- 导入表空间
impdp system/1qaz2wsx directory=dump_dir dumpfile=tablespace.dmp tablespaces=tbsp_1
-- 导入全数据库
impdp system/1qaz2wsx directory=dump_dir dumpfile=fulldatabase.dmp full=y
```

## IMPDP命令参数
- REMAP_SCHEMA：remap_schema=source_schema:target_schema
- REMAP_TABLESPACE：remap_tablespace=source_tablespace:target_tablespace
- SQLFILE：sqlfile=[directory_object:]file_name 从DMP文件中提取对象的DDL语句
- TABLE_EXISTS_ACTION：table_exists_action={skip | append | truncate | replace}
  + skip：导入作业会跳过已存在表处理下一个对象。
  + append：会追加数据。
  + truncate：导入作业会截断表，然后为其追加新数据。
  + replace：导入作业会删除已存在表，然后重建表追加数据。

## SQL* Loader概述
SQL*Loader可以将外部文件中的数据加载到Oracle数据库中，使用SQL*Loader导入数据时，必须编辑一个控制文件（.CTL）和一个数据文件（.DAT）
- 自由格式加载
```sql
load data
  infile 'd:\data\student.txt'
  into table student
-- 加载工具通过position控制数据的分割
  (stuno position(01:04) integer external,
   stuname position(11:14) char,
   sex position(21:22) char,
   old position(29:30) integer external
  )
```
- 固定格式加载数据
```sql
load data
infile 'd:\data\persons.csv'
append into table persons -- 数据的加载方式为append，表示在表中追加新数据
fields terminated by ','  -- 指定数据文件中的分隔符为逗号“，”
(code,name,sex,old)
```
