## 分支结构
- IF文
```VBA
Sub IfSample()
    Dim score As Integer
    score = 59

    If score >= 60 Then
        If score >= 90 Then
            MsgBox "Perfect!!!"
        ElseIf score >= 80 Then
            MsgBox "nice"
        Else
            MsgBox "good"
        End If
    Else
        MsgBox "Not Good..."
    End If
End Sub
```

- CASE文
```VBA
Sub CaseSample()
    Dim score As Integer
    score = 85

    Select Case score
        Case 90 To 100
            MsgBox "Perfect!!!"
        Case 80 To 89
            MsgBox "Nice"
        Case 70 To 79
            MsgBox "Good"
        Case Else
            MsgBox "Not Good"
    End Select
End Sub
```
