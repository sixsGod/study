### 变量
- 定义变量
```VBA
REM Dimension 变量名
Dim name As String
Dim age As Integer
```

- 使用变量
```VBA
name = "Allen"
age = 18
    
Debug.Print (name)
Debug.Print (age)
```

### 数据类型
```
REM Dim 变量名 As 数据类型
Dim name As String
Dim name As Variant ' 自适应数据类型
Dim flg As Boolean
Dim age As Byte
Dim year As Integer, month As Integer
Dim age As Long
Dim age As Single
Dim age As Double
Dim days As Date
```
